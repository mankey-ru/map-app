# VanillaForum

Installation:
```sh
git clone <repo>
npm install
```

Run webpack dev server with Hot Module Reloading:
```sh
npm run dev --harmony_generators
```

Run "plain" express server:
```sh
npm start
```

Build client-side JS:
```sh
npm run build
```

- Webpack
- Vue.js
